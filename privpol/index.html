Vertex Bot - Privacy Policy

1. Introduction

This Privacy Policy explains how Vertex Bot ("the Bot") collects, uses, and protects information from users. By using the Bot, you agree to the practices described in this policy.

2. Data Collection

Vertex Bot collects only minimal information necessary to function, including:

  • Discord server IDs and channel IDs
  • User IDs for command tracking and moderation
  • Command usage data for functionality and analytics

No sensitive personal information such as passwords or payment details is collected.

3. How Data is Used

  • To provide bot functionality and respond to commands.
  • To improve bot features and performance.
  • To moderate and manage servers if moderation commands are used.

4. Data Sharing

Vertex Bot does not share or sell your data to third parties. Data may be used internally by the development team for maintenance, troubleshooting, and analytics.

5. Data Retention

Collected data is stored only as long as necessary for the Bot to function and may be deleted upon server removal or request.

6. Security

Vertex Bot implements standard security measures to protect data. However, no system is completely secure, and the Bot cannot guarantee absolute protection.

7. User Rights

You may request deletion of your data or information associated with your server by contacting the Vertex development team via Discord.

8. Changes to Privacy Policy

Vertex Bot may update this Privacy Policy periodically. Users will be notified of significant changes via Discord or the bot's documentation.

9. Contact

If you have questions regarding this Privacy Policy, please contact the Vertex Bot development team via Discord.
